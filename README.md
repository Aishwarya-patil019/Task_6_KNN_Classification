# Task_6_KNN_Classification

## Objective
Implement K-Nearest Neighbors (KNN) algorithm for classification using the Iris dataset.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
1. Loaded Iris dataset
2. Removed unnecessary Id column
3. Split data into training and testing sets
4. Normalized features using StandardScaler
5. Trained KNN model
6. Tested different K values
7. Evaluated accuracy
8. Generated confusion matrix
9. Visualized decision boundaries

## Result
The KNN model successfully classified Iris flower species with very high accuracy.
